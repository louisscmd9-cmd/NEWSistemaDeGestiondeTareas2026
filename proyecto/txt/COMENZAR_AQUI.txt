⚙️ CONFIGURAR AUTOMATIZACIÓN DIARIA (Opcional):
═══════════════════════════════════════════════════════════════════════════════

WINDOWS - Task Scheduler:
1. Abre: Programador de tareas (Task Scheduler)
2. Crea tarea básica
3. Trigger: Diario a las 00:00 (medianoche)
4. Acción: 
   - Programa: cmd.exe
   - Argumentos: /c node create-recurrent-tasks.js
5. Carpeta: c:\...\proyecto

LINUX/MAC - Cron:
1. Terminal: crontab -e
2. Agrega: 0 0 * * * cd /ruta/proyecto && node create-recurrent-tasks.js
3. Guarda


🚨 SI ALGO FALLA:
═══════════════════════════════════════════════════════════════════════════════

Error: "Node no se reconoce"
→ Instala Node.js desde: https://nodejs.org

Error: "Puerto 3000 en uso"
→ Cambia PORT en app.js
→ O cierra proceso que usa puerto 3000

Error: "Base de datos bloqueada"
→ Cierra otros procesos del proyecto
→ Reinicia el servidor

Error al Editar/Eliminar tareas
→ Verifica credenciales de jefe
→ Asegúrate que sea quien creó la tarea
