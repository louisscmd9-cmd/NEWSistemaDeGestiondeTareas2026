⚙️ CONFIGURAR AUTOMATIZACIÓN (Opcional)
═════════════════════════════════════════════════════════════════════════════════

Para que las tareas se dupliquen AUTOMÁTICAMENTE cada día:

WINDOWS - Task Scheduler:
   1. Programador de tareas
   2. Crear tarea básica
   3. Trigger: Diario a las 00:00
   4. Acción: cmd.exe /c node create-recurrent-tasks.js
   5. Carpeta: c:\...\proyecto

LINUX/MAC - Cron:
   1. crontab -e
   2. 0 0 * * * cd /ruta/proyecto && node create-recurrent-tasks.js
   3. Guardar

🚨 SOLUCIÓN DE PROBLEMAS
═════════════════════════════════════════════════════════════════════════════════

Error: "Node no se reconoce"
   → Instala Node.js: https://nodejs.org

Error: "Puerto 3000 ya está en uso"
   → Cambia PORT en app.js
   → O cierra otro proceso en ese puerto

Error: "No puedo editar tarea"
   → Verificar: ¿Eres el jefe que la creó?
   → Verificar: ¿La tarea está pendiente?

Tarea no se copia automáticamente
   → Ejecutar manualmente: node create-recurrent-tasks.js
   → Verificar: ¿Está marcada como recurrente?
   → Verificar: ¿Era estado pendiente?
